# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] JWT Authentication
- [x] Login(JWT)
- [x] Create Account
- [x] Edit Profile
- [x] See Profile
- [x] Add/Room From Favourites
- [x] See Favs
- [x] List Rooms
- [x] See Room
- [ ] Search By Coords
